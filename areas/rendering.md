# Rendering

**Scope:** SDL2 rendering of all game elements, camera system, UI.

**Files:** `src/systems/renderer.nim`, `src/systems/camera.nim`

**Description:**
Renders backgrounds, platforms (gray rectangles), characters (colored rectangles with squash/stretch), hazards (red), exits (character-colored outlines), buttons, doors, and UI overlay (character bar, narration text). Smooth-follow camera centered on active character.
